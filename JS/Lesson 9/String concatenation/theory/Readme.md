# Конкатенация строк
***
### Оператор + для конкацинации строк
```'Hello' + 'World'```
###  Переменные в конкацинации строк
```
const hello = 'Hello'
const world = 'World'

const greeting = hello + '' + world
```
### Шаблонные строки
```
const hello = 'Hello'
const world = 'World'

const greeting = ${hello} ${world}
```
