# Операторы
***
> Оператор - это встроенная функция

```Операторы```
> Фрифмитические + - * /

> Сравнения === !== <= >=
 
> Логические ! && ||
 
> Присваивания =
> ***
## Текстовые операторы
```typeof``` ```instanceof``` ```new``` ```delete```
## Пример
```
let a, b
a = 10
b = a

let c = a + b

console.log(c) // 20
```
***
## Унарные операторы
> у унарных операторов всегда один операнд (аргумент)
```
a++
+a
delete obj.a
typeof a
new Object()
```
## Бинарные операторы
> у бинарных операторов два операнда (аргумента)
```
a = 5
a + b
a += 5
a === b
a && b
```
***
## Форматы записи операторов
### Инфиксная запись
```
a = true
a + b
a += 5
a || b
a > b
```
### Префиксная запись
```
++a 
delete obj.a
typeof a
```
### Постфиксная запись
```
a++
myFunction()
```
